# 🔗 Link Tree

Um projeto simples de **Link Tree** desenvolvido com **HTML e CSS puro**, com o objetivo de centralizar meus links e informações em uma única página.

## 🚀 Tecnologias

- HTML5
- CSS3

## 💡 Sobre o projeto

Este projeto funciona como uma página pessoal de links, similar a um **link-in-bio**, onde é possível acessar rapidamente meus perfis, contatos e outras informações importantes.

O foco foi criar um layout limpo, responsivo e visualmente agradável utilizando apenas recursos nativos da web, sem uso de frameworks ou bibliotecas externas.

## 🎯 Funcionalidades

- Layout responsivo
- Organização de links principais
- Estilização moderna com CSS
- Estrutura simples e fácil de manter

## 📂 Como usar

1. Clone o repositório:

```bash
git clone https://github.com/mathgoms02/link-tree.git
```

2. Acesse a pasta do projeto:

```bash
cd link-tree
```

3. Abra o arquivo `index.html` no navegador.

## 📌 Objetivo

Este projeto foi desenvolvido principalmente para praticar e consolidar conceitos de **CSS**, desde fundamentos básicos até técnicas mais avançadas, aplicados em um cenário real.
